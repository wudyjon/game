# 🍺 酒桌小游戏

一个精美的网页版酒桌小游戏合集，适合朋友聚会、公司团建、家庭聚餐时活跃气氛。

## 🎮 包含游戏

### 🎯 幸运转盘
- 输入玩家名字，转盘随机选择一位"幸运儿"
- 支持动态添加/重置玩家
- 流畅的转盘动画

### 💣 数字炸弹
- 设定数字范围，系统随机生成一个"炸弹数字"
- 玩家轮流猜数字，范围不断缩小
- 猜中炸弹数字的人接受惩罚

### 🎭 真心话大冒险
- 内置丰富的真心话和大冒险题库
- 支持选择真心话、大冒险或随机抽取
- 持续更新更多有趣题目

### 🎲 掷骰子
- 支持 1-6 颗骰子同时投掷
- 精美的骰子动画效果
- 自动计算总和

### 🎲 随机选择
- 不知道玩什么？让系统随机推荐一个游戏

## 🚀 如何使用

### 方法一：直接打开
下载项目后，直接用浏览器打开 `index.html` 文件即可游玩。

### 方法二：本地服务器
```bash
git clone https://github.com/YOUR_USERNAME/drinking-games.git
cd drinking-games
# 使用 Python 启动本地服务器
python -m http.server 8080
# 或 Node.js
npx serve .
```
然后访问 `http://localhost:8080`

## 📱 技术栈

- HTML5
- CSS3 (响应式设计，支持移动端)
- Vanilla JavaScript (无依赖)

## 🖼️ 预览

项目采用深色主题设计，配有精美的动画效果，在手机和电脑上都有良好的体验。
